# Changelog

All notable changes to the Qlik Development Framework will be documented in this file.

## [0.1.0] - 2026-03-01

Initial scaffold release.

### Added

- Directory structure for `.claude/` framework (agents, skills, hooks)
- 7 agent definitions with YAML frontmatter (requirements-analyst, data-architect, script-developer, expression-developer, viz-architect, qa-reviewer, doc-writer)
- 13 skill definitions with YAML frontmatter and supporting placeholder files
- Hook configuration in `.claude/settings.json` with QVS syntax validation hook
- `inputs/` directory structure for user-provided source materials
- `artifacts/` directory structure for framework-generated output
- README, LICENSE (MIT), NOTICE, CONTRIBUTING, CHANGELOG
